Information about the Test: For this Visualization Experiment you can choose either Group A or Group B. You have to click on any of them.

Group A: If you click on this option you will see 8 x 8 color grid. The timer will start and and from this grid you have to choose unique color and click on it for success. After success you will see 'next' button from which you can go to next run of the experiment.

Group B: If you click on this option you will first see flood fill color grid of same color for 5 seconds and then 'next' option will visible. Click on it it will land you to next page which will have 8 x 8 color grid. The timer will start and from this grid you have to choose unique color and click on it for success. After success you will see 'next' button from which you can go to next run of the experiment.
